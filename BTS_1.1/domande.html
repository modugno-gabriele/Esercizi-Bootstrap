A cosa serve bootstrap?

Bootstrap è un framework front-end open-source estremamente popolare e utile nel campo dello sviluppo web. Il suo scopo principale è semplificare e velocizzare la creazione di siti web moderni e responsive

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Una CDN (Content Delivery Network) è una rete di server distribuiti che memorizza i contenuti statici (come i file di Bootstrap). Vantaggi del CDN: Garantisce maggiore velocità di caricamento (servendo i file dal server più vicino all'utente), riduce il carico sul tuo server principale e aumenta l'affidabilità e la sicurezza (es. contro attacchi DDoS).

A cosa serve questo attributo del tag meta?

l'attributo content="width=device-width, initial-scale=1" del tag meta serve a rendere il sito web responsive.

Cosa devi importare per creare una pagina web con bootstrap

devi usare <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css">

Che differenza c’è tra Container e Container Fluid

La classe .container crea un contenitore a larghezza fissa responsiva e centrata.



La classe .container-fluid crea un contenitore a larghezza fluida al 100%. Si estende sempre da bordo a bordo della finestra del browser
